什么是庭院别墅景观设计

庭院别墅景观设计是植百汇为别墅、私家庭院提供的景观设计、施工与长期养护服务，涵盖中式、日式、现代等风格的植物配置、软硬景与花园营造，设计施工一体化，落地更可控。室内与露台的植物景观可一并参考豪华住宅 / 四代宅绿植景观。

适合哪些客户

  • 别墅、独栋、带院住宅业主。
  • 想做花园 / 庭院景观、需要设计 + 施工落地的家庭。
  • 希望长期有人养护打理庭院的客户。
  • 对风格（中式 / 日式 / 现代）有明确要求的业主。

服务流程

上门勘测 / 量房 → 设计方案 + 效果图 → 报价 → 施工落地 →（可选）长期养护。

价格怎么算

按庭院面积与设计方案报价，构成 = 设计费 + 苗木 / 材料 + 施工 +（可选）养护。可免费上门勘测，出设计方向与预算区间。
